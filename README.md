# REPO 002: Advanced Crypto Matrix (Legacy to PQC)
This repository contains a full spectrum of cryptographic assets for CBOM scanning tests.

## Targets:
- **Legacy**: MD5, SHA-1, DES, RC4, RSA-1024, secp160k1.
- **Modern**: AES-256, SHA3, SHAKE.
- **PQC**: ML-KEM, ML-DSA, BIKE, HQC, NTRU, MAYO, SNOVA.
